### 🍱 Nền tảng chuyển đổi số ngành F&B tích hợp AI

Hệ sinh thái web toàn diện cho một doanh nghiệp trong ngành ẩm thực — từ trang thông tin thương hiệu, hệ thống hỗ trợ khách hàng đa kênh, đến bảng điều khiển quản lý vận hành nội bộ.

**Vai trò:** Fullstack Developer — phụ trách toàn bộ frontend (Next.js), xây dựng backend API (Node.js), tích hợp AI Agent và phát triển hệ thống realtime.

| Giải pháp kỹ thuật | Mô tả |
| :--- | :--- |
| **Hệ thống Realtime** | Bảng điều khiển nhân viên với quản lý hàng đợi, xử lý trạng thái kết nối bằng SignalR, thông báo đa tầng (Desktop / Audio / Visual) |
| **Tích hợp AI Agent** | Trợ lý AI đa ngôn ngữ (RAG), xử lý hình ảnh (multimodal), trích xuất dữ liệu hóa đơn, tự động gửi thông báo qua Telegram/Email |
| **Tối ưu hiệu năng** | Bento UI hiện đại, tối ưu hình ảnh + caching, PageSpeed > 90 trên mọi thiết bị |

**Kết quả đạt được:**
- Tự động hóa ~60-70% yêu cầu thường gặp của khách hàng
- Giảm thời gian phản hồi từ phút xuống còn giây
- Xóa bỏ rào cản ngôn ngữ cho khách quốc tế

**Tech stack:** `Next.js` · `React` · `Node.js` · `SignalR` · `AI RAG` · `MongoDB` · `Redis` · `Docker`

Trang hồ sơ năng lực cá nhân được xây dựng theo phong cách Bento Grid, hỗ trợ đa ngôn ngữ và đạt điểm hiệu năng gần tuyệt đối.

**Vai trò:** Thiết kế & phát triển toàn bộ — từ concept UI đến triển khai production trên Vercel.

| Giải pháp kỹ thuật | Mô tả |
| :--- | :--- |
